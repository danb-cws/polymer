Github commit widget

A demo component using Google Polymer, querying the Github api to show recent commits for any public repository.

This demo shows an example of using Polymer to create a web component with simple markup, where the complexity is abstracted away to an externally located resource. Here's how it works:

  • The webcomponents polyfill script is included in the head of the page.
  • The custom component is imported using a link rel="import" in the document head: 
                            <link rel="import" href="https://fiery-torch-7498.firebaseapp.com/github-commits.html">
  • Each of the "github-commits" instances in this page are marked up as: 
                            <github-commits>
                            <span class="user">angular</span>
                            <span class="name">angular.js</span>
                        </github-commits>
  • All of the logic and presentation around the "github-commits" element is contained in the imported component, including css, script and markup. (View source of github-commits.html for full code).
